

# To-Do App

## Prerequisites
- Node v16

## Usage

Run the following command to get started:

```sh
yarn dev:init
```

To start a development server (make sure to run the above command first):

```sh
yarn dev
```

To make and start a build use:

```sh
yarn build && yarn start
```

To run the tests:

```sh
yarn test
```
